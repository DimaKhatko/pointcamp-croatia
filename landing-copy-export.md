# PointCamp Croatia — Landing copy export

All user-visible Ukrainian text on the landing page, in page order (per
`src/routes/index.tsx`).

**Where strings live:**
- Most section **headings / eyebrows / leads / button labels** are **hardcoded in JSX** in each section's `.tsx`.
- **Repeated/list data** (cards, schedule, activities, pricing lists, FAQ, quick facts) lives in **`src/components/landing/data.ts`** and is rendered by the components.
- **Photo `alt` text** is not visible on screen but is surfaced to screen readers / SEO; listed per section, marked _(alt)_.
- Each string below is tagged with its role and its source location.

Legend: `[JSX]` = hardcoded in the component file · `[data.ts]` = in `src/components/landing/data.ts`.
